2. Git: **[Link](https://git-scm.com)**

## Setting
1. Run `git clone https://github.com/HELLSNAKES/Music-Slash-Bot.git`
2. Run `cd music-slash-bot`
3. Run `npm i` to install the modules required
4. Rename `.env_example` to `.env`
5. Edit `.env`
